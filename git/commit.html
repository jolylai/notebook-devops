<Footer>

# 示例

<!-- header -->
feat: 添加了分享功能

<!-- body -->
给每篇博文添加了分享功能

- 添加分享到微博功能
- 添加分享到微信功能
- 添加分享到朋友圈功能

# Header ⭐️

修改类型

  • feat：新功能（feature）
  • fix：修补 bug
  • doc：文档（documentation）
  • style： 格式（不影响代码运行的变动）
  • refactor：重构（即不是新增功能，也不是修改 bug 的代码变动）
  • test：增加测试
  • chore：构建过程或辅助工具的变动
  • perf：性能优化，提高性能的代码更改

也可使用Gitmoji添加 emoji

# Body

Body 部分是对本次 commit 的详细描述，可以分成多行。

# Footer

Footer 部分只用于两种情况：

  • 关联 Issue
  • 关闭 Issue
